# 💻 [2026-01] 나만의 C언어 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* 나의 은행 계좌 관리 프로그램

<br> 


## 2. 시나리오 (5~10줄)
* (이곳에 한 학기 동안 개발할 프로그램의 전체 시나리오를 개괄적으로 요약합니다.)
*  이 프로그램은 사용자의 은행 계좌 잔액을 관리해주는 프로그램 입니다. 시스템은 사용자의 이름, 계좌번호, 입출금 금액, 이자율 등을 바탕으로 계좌에 남아있는 잔액을 계산해줍니다. 또한, 사용자의 계좌가 여러개일 경우 계좌별로 따로 잔액을 관리하고 다른 이자율을 적용하며, 계좌를 선택할 수 있고, 잔액보다 큰 금액을 출금할 시 잔액부족 메세지가 뜨는 프로그램으로 완성할 계획입니다. 

<br>


## 3. 예상 기능 및 메뉴 (최소 5개)
1)계좌 정보 입력
2)입금
3)출금
4)이자 적용
5)잔액 조회, 잔액부족 알림
6)프로그램 종료

# 🚀 [버전별 개발 일지 & AI 협업 기록]

## 🟦 [1차 과제: V1.0] 기초 뼈대 구축 (입출력과 변수)
### **✨V1.0 개발 내용:**
  * 사용자로부터 계좌정보, 입출금 금액, 이자율을 입력받아 계좌의 잔액을 조회하는 프로그램입니다. 사용자의 이니셜(문자), 계좌번호와 입출금 금액(정수), 이자율(실수)를 입력받는 화면을 구현하였고 사칙연산을 활용해 잔액을 계산하여 출력했습니다.
    
### **🤖 AI 파트너십 과정**
 1. **내용 1**
프롬프트 요약 : 4단계 (변수 입출력-조건문-반복문/메뉴-함수/배열)로 프로그램을 만들 수 있는 쉬운 난이도의 프로그램을 추천 요청
적용 내용 : 간단한 은행 계좌 관리 프로그램을 만들어 여러 기능을 넣어야겠다고 생각함.
      
 3. **내용 2**
프롬프트 요약 : 변수 5개 이상, 자료형 3개 이상, 산술연산 1개 이상이 포함된 프로그램에 대한 논의
적용 내용 : 입출금에 대한 계산뿐만 아니라 이자율(실수형)을 적용해서 프로그램을 만들 수 있게 됨.
    
### **🛠️ Troubleshooting & 기술 회고:**
 문제 1) 컴파일 프로그램에서 오류가 나서 실행이 안되는 상황이 발생했는데, AI에게 오류가 나는 이유를 질문하고 input칸에 숫자를 미리 입력해서 해결함.

문제 2) scanf(" %c", initial)의 %c앞에 공백을 띄우지 않아서 엔터키로 인식하는 상황이 발생해서 공백을 두어 코드를 수정함.

문제 3) 처음 계좌 잔액을 0으로 두지 않고 실행했더니 계산한 값에 오류가 나서 AI와 상의 후 처음 계좌 잔액을 0으로 설정함.
